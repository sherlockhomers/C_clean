# CleanC — 产品需求文档 (Web版)

## 1. 产品概述

CleanC 是一款面向普通用户的 Windows C盘深度清理与空间管理工具，内置 AI 智能助手。本 Web 版本为 UI 原型展示，完整还原桌面端的视觉设计和交互模式。

- 核心定位：让用户一眼看清C盘状况，一键完成清理和迁移
- 目标用户：C盘空间不足、电脑卡顿的 Windows 普通用户
- 设计风格：Forge Tools 风格，橙色系主色调 + AI 紫蓝辅助色，现代卡片式布局

## 2. 核心功能

### 2.1 功能模块

1. **仪表盘**：C盘使用率环形图、可释放空间统计、AI精要摘要、待处理项目列表、健康评分
2. **快速清理**：安全清理项列表、全选/取消、一键清理确认弹窗
3. **占用侦探**：软件/文件夹/文件类型排名、时间线分析、隐藏占用检测
4. **深度扫描**：大文件扫描、重复文件检测、卸载残留扫描
5. **软件迁移**：已安装软件列表、迁移向导、兼容性评估
6. **路径迁移**：系统文件夹迁移、软件输出路径迁移、AI推荐排序
7. **AI 助手**：对话界面、快捷指令面板、Markdown渲染
8. **空间监控**：历史趋势图、定时清理计划、告警设置
9. **设置**：通用设置、AI配置、暗色模式切换
10. **关于**：版本信息

## 3. 核心流程

用户打开应用 → 仪表盘自动展示C盘概览和AI摘要 → 用户选择操作：
- 一键清理：选择清理项 → 确认弹窗 → 执行清理 → 结果报告
- 占用侦探：查看排名 → 展开详情 → 执行操作
- 软件迁移：选择软件 → 向导式迁移 → 完成
- AI助手：自然语言对话 → AI分析建议 → 确认执行

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：橙色 `#FF6B35` / 渐变 `#FF6B35 -> #FF8F5E`
- **AI辅助色**：渐变紫蓝 `#7C3AED -> #3B82F6`
- **背景色**：主区域 `#FAFAFA` / 卡片 `#FFFFFF` / 侧边栏 `#FFFFFF`
- **暗色模式**：主背景 `#1E1E1E` / 卡片 `#2D2D2D` / 侧边栏 `#252526`
- **卡片样式**：圆角 12px，浅阴影
- **风险标签**：安全=绿色 / 警告=橙色 / 危险=红色，胶囊形
- **按钮**：主操作橙色渐变圆角 / AI操作紫色渐变圆角
- **侧边栏**：220px宽度，支持收起至60px
- **字体**：标题 18px bold / 正文 14px / 辅助 12px

### 4.3 响应式设计

桌面优先设计，最小宽度 1024px，侧边栏可收起适配更小屏幕。

## 5. 急救模式

当C盘可用空间 <5% 时自动激活：
- 界面变为红色调警告风格
- 三步急救流程（立即释放 → 安全释放 → 建议释放）
- AI自动弹出引导用户
